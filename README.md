# EC5MAP

**Интерактивная карта школы** — учебный проект 10 класса, разработанный на движке Godot. Позволяет пользователям взаимодействовать с картой учебного заведения и осуществлять поиск кабинетов.

## 🛠 Технологии
- **Godot Engine** — среда разработки (движок).
- **GDScript** — скриптовый язык программирования.

## 📁 Структура проекта
```
EC5MAP/
├── core/ # Основные скрипты и логика приложения
├── scenes/ # Сцены Godot (интерфейс, карта, элементы)
├── scripts/ # Дополнительные скрипты
├── images/ # Изображения для карты и интерфейса
├── logos/ # Изображения для экспорта на Android
├── other/ # Шрифты и стили
├── project.godot # Главный файл проекта Godot
├── export_presets.cfg # Настройки экспорта проекта
└── .gitattributes/.gitignore
```
